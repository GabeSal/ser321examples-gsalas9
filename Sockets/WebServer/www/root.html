Welcome to the Java WebServer

Supported GET Requests

  • /random – Loads the image viewer page (index.html)
  • /json – Returns a random image and label as JSON
  • /file/sample.html – Attempts to return the contents of sample.html
  • /multiply?num1=3&num2=4 – Multiplies two numbers → Example
  • /github?query=users/amehlhase316/repos – Fetches and displays GitHub repository data → Example
  • /reverse?text=hello – Reverses a string → Example
  • /convert?temp=100&type=ftoc – Converts Fahrenheit to Celsius → Example
  • /convert?temp=37&type=ctof – Converts Celsius to Fahrenheit → Example

📁 Files in the www/ directory

You can use /file/www/FILENAME to access files directly. Example: /file/www/index.html
